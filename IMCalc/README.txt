Bem-vindo ao meu projeto de cálculo de Índice de Massa Corporal (IMC)!

Este projeto é uma ferramenta simples que permite aos usuários calcular seu IMC com base em seu peso e altura. O IMC é uma medida da gordura corporal comummente usada para avaliar o estado nutricional de adultos.

Para executar este projeto, é necessário instalar a biblioteca customtkinter. Esta biblioteca fornece widgets personalizados para a criação de interfaces gráficas de usuário (GUIs) no Python, e é necessária para a execução deste projeto.

Para instalar a biblioteca criei um arquivo requirements.txt que todas as bibliotecas que usamos neste projeto, para executar o requirements execute o seguinte comando no terminal:

pip install -r requirements.txt

Uma vez que a biblioteca esteja instalada, você pode executar o projeto simplesmente executando o arquivo "software.py". A GUI irá guiá-lo através do processo de entrada de seus dados de peso e altura, e calculará automaticamente o seu IMC.
